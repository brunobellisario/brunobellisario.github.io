Fish diversity of Mediterranean seagrasses

  

Download
DATASET

In this excel file you can find the data extracted from an extensive literature review about the presence of 248 fish species on native seagrasses in the Mediterranean Sea. This dataset provides a full list of the fish species recorded on seagrasses by present literature, reporting each observation associated with: i) Mediterranean ecoregion, ii) the reference FAO zone and iii) GFCM Area, iv) site and v) geographic location, vi) season, vii) year and viii) sampling method, ix) habitat composition and x) substrate, xi) life stage, xii) latitude and xiii) longitude of each location and xiv) bibliographic information.

The present dataset is associated with the following paper:
Lattanzi A, Bellisario B, Cimmaruta R (2023) A review of fish diversity in Mediterranean seagrass habitats, with a focus on functional traits - bioRxiv 2023.09.23.559097 - https://doi.org/10.1101/2023.09.23.559097

Bird ringing data from Ponza Island + traits
& EURING codes

Download
DATASET - R CODE

Dataset accompanying the paper Co-migration fidelity at a stopover site increases over time in African-European migratory landbirds. The dataset includes individual catch data (Sheet1), EURING codes (Sheet2) and traits (Sheet3) of 31 most abundant non-resident species landing on the stopover area of Ponza over a period of 15 years (2007 to 2021), during which the procedures were fully standardized.

The R code is part of the same paper and is divided in three main functions allowing to replicate the results:
1. ring2matrix - converts a time-list of bird ringing data in a temporal matrix representing the number of individuals x species x time unit.
2. tempnet - network inference from time series data.
3. dist_fun - compute the pairwise co-migration fidelity from a module x time structured-table by means of the normalized mutual information.

Mediterranean seagrass amphipods

Download
DATASET - R CODE

Dataset in excel format including:
1. Distributional data
2. Traits
3. Environmental characterization of study localities
4. Least-cost distance matrix

Part of the data used in this meta-analysis have been previously published as open access in GBIF (AxIOM: Amphipod crustaceans from insular Posidonia oceanica seagrass meadows - https://www.gbif.org/dataset/b146a93c-657b-4768-aa51-9cabe3dac808 - DOI10.15468/nnjtke).
This data set includes occurence/presence but also densities and associated metadata.
Other source of data can be found here: http://www.vliz.be/en/imis?module=dataset&dasid=6658

R code to analyze the taxonomic, functional and phylogenetic beta diversity of Mediterranean seagrass amphipods. The code provides an easy way to access the results obtained in:

Bellisario B, Camisa F, Abbattista C, Cimmaruta R (2021) Determinants of different facets of beta diversity in Mediterranean marine amphipods. Marine Biology 168, 90 - https://doi.org/10.1007/s00227-021-03905-z

Part of dataset has also been used in the following publication:

Bellisario B, Camisa F, Abbattista C, Cimmaruta R (2019) A network approach to identify bioregions in the distribution of Mediterranean amphipods associated with Posidonia oceanica meadows. PeerJ 7:e6786 - https://doi.org/10.7717/peerj.6786
